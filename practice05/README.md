# Practice 05: LLM Chat with Skill System

## 功能描述

本项目实现了一个基于 LLM 的聊天系统，具有以下功能：

1. **技能系统**：
   - 自动读取 `.agents` 目录下的技能定义
   - 支持从 SKILL.md 文件中提取技能信息
   - 能够加载技能正文内容并通过系统提示发送给 LLM

2. **通知技能**：
   - 用于撰写、修改、润色通知
   - 通知不能以"通知"二字开头，必须冠以"XX部"前缀
   - 用户未提供部门时，统一用"XX部"代替

3. **聊天历史管理**：
   - 每五次聊天提取一次关键信息（5W规则）
   - 将关键信息保存到本地文件 `D:\chat-log\log.txt`
   - 自动压缩聊天历史，保持上下文长度在合理范围内

4. **工具调用**：
   - 支持多种工具，如文件操作、网页访问、文档仓库查询等
   - 能够处理工具调用的结果并生成自然语言响应

```

```

## 技能定义格式

技能定义文件 `SKILL.md` 应包含以下内容：

```yaml
---
name: 技能名称
description: 技能描述
---
技能正文：
技能详细说明和使用规则
```

## 测试方法

1. **测试技能系统**：
   ```bash
   python test_skills.py
   ```

2. **测试通知技能**：
   ```bash
   python test_notice_skill.py
   ```

3. **运行聊天系统**：
   ```bash
   python chat_with_summary.py
   ```

## 配置说明

项目使用 `.env` 文件进行配置，主要配置项包括：

- `LLM_BASE_URL`：LLM API 基础 URL
- `LLM_MODEL`：使用的模型名称
- `LLM_API_KEY`：API 密钥
- `LLM_TEMPERATURE`：温度参数
- `LLM_MAX_TOKENS`：最大 tokens 数
- `ANYTHINGLLM_API_KEY`：AnythingLLM API 密钥
- `ANYTHINGLLM_WORKSPACE_SLUG`：AnythingLLM 工作区 slug
